# 📊 Cotação Manager

Plugin WordPress para gerenciamento de cotações com painel administrativo, histórico de alterações e exibição via shortcode.

---

## 🚀 Funcionalidades

- 📌 Painel próprio no admin (menu lateral)
- 💰 Cadastro de preços (soja, milho, trigo)
- 🧠 Máscara automática de moeda (R$)
- ✅ Validação e sanitização dos dados
- 🕓 Histórico automático de alterações
- 🗑️ Exclusão do histórico
- 🌐 Shortcode para exibição no frontend

---

## ⚙️ Como usar

### 1. Ativar o plugin
- Vá em **Plugins → Ativar Cotação Manager**

### 2. Inserir cotações
- Acesse o menu **Cotação** no painel
- Preencha os valores e salve

### 3. Exibir no site
Use o shortcode:
[cotacao]

---

## 📌 Observações

- Os dados são armazenados no banco via `get_option`
- O histórico é salvo automaticamente a cada alteração
- Não depende de ACF ou plugins externos
